# OXWEB — статическая версия для GitHub Pages

Это максимально простой вариант для загрузки через GitHub UI без папки `assets`.

В корне репозитория должны лежать:

- `index.html`
- `index-DNPEzTbD.js`
- `index-BTghxb0R.css`
- `.nojekyll`
- `README_UPLOAD.md`

GitHub → Settings → Pages:

- Source: Deploy from a branch
- Branch: main
- Folder: /root

Если сайт не обновился сразу, подождите 1–3 минуты и обновите страницу с Ctrl+F5.
